# 4. 날짜와 타임스탬프 다루기

---

- 날짜/시간 함수
- 날짜 자료형
- 타임스탬프 자료형
- 문자열 함수



## 현재 날짜와 타임스탬프 추출하기

```sql
SELECT
	CURRENT_DATE() AS dt,
	CURRENT_TIMESTAMP() AS stamp
;
```



## 지정한 값의 날짜/시각 데이터 추출하기

```sql
SELECT
	-- SparkSQL: 'CAST(value AS type)'
	CAST('2023-01-17' AS date) AS dt,
	CAST('2023-01-17 23:28:00' timestamp) AS stamp
;
```
